Consent Management Flow

How we handle consent at Roofwander with Google Consent Mode v2.

User visits website
↓
Cookie Banner (Accept / Reject / Customize)
↓
User choice saved(user_consent_choice cookie)
↓
ConsentContext updates state
↓
updateGTMConsent() triggered
↓
Google Tag Manager receives consent signals
↓
Consent Mode v2analytics_storage / ad_storage
↓
Tags activate conditionally • GA4 analytics • Marketing tags • Advertising pixels
