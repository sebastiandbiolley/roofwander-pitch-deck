## How to Edit Posts

1. Open the `linkedin-post.md` file in any text editor (VS Code, Notepad, etc.)
2. Edit the content in Markdown format
3. Copy the text to LinkedIn when ready to post (LinkedIn supports basic formatting from pasted Markdown)

## How to Open HTML Diagrams

1. Navigate to the topic folder (e.g. `tracking-flow/`)
2. Double-click the `*.html` file, or right-click в†’ **Open with** в†’ **Browser** (Chrome, Edge, Firefox)
3. The diagram will render in your default browser

## How to Export HTML to PDF

1. Open the HTML diagram file in your browser
2. Press **Ctrl+P** (Windows) or **Cmd+P** (Mac) to open the Print dialog
3. In the destination dropdown, select **Save as PDF**
4. Click **Save** and choose where to save the file
5. Upload the PDF to LinkedIn for document/carousel posts

**Tips for best results:**
- Use landscape orientation if the diagram is wide
- Ensure "Background graphics" is enabled in print settings so shadows and colors are preserved
- Preview the PDF before posting to check layout
